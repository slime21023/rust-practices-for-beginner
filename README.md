# Rust 程式設計入門與實踐

歡迎來到 Rust 程式設計入門與實踐！本專案旨在為有經驗的開發者提供一個系統性的 Rust 學習資源，特別適合那些已經熟悉其他程式語言但想要快速掌握 Rust 的開發者。

- 有經驗的開發者，但對 Rust 是初學者
- 想要快速複習 Rust 核心概念的開發者
- 希望通過實際範例學習 Rust 的學習者
